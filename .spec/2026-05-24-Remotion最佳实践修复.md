## 修复结果

### P0 — 渲染失效 (3/3 ✅)
- [x] **PricingCard `animate-[shimmer]`** — 改用 `useCurrentFrame()` + `interpolate()` 驱动 `transform: translateX()`，120 帧循环
- [x] **AuroraBg 动态定位** — `top-[calc(20%+${y}px)]` / `left-[calc(10%+${x}px)]` → `style={{ top, left }}`
- [x] **DataBarChart/Cursor/CoverflowCarousel 动态值** — 全部 `${...}` 模板变量从 Tailwind class 迁移到 `style={{}}`（共 6 处）

### P1 — 偏离最佳实践 (3/3 ✅)
- [x] **ElementRenderer `Video` 导入** — `from "remotion"` → `from "@remotion/media"`
- [x] **TransitionSeries 预挂载** — 重构为 `<TransitionSeries>` 替代手动 `<Sequence>` 叠加，TransitionSeries 内置预挂载
- [x] **AnimatedBar `transition-all`** — 直接删除无效 class

### P2 — 架构债务 (1/1 ✅)
- [x] **`@remotion/transitions` 集成** — `transitionRegistry.ts` 重写为真实 registry（fade/slide/wipe），`TemplateRenderer.tsx` 重构为 `<TransitionSeries>`，`calculateTotalFrames()` 适配 TransitionSeries 的 duration 计算

### P3 — 清理 (2/2 ✅)
- [x] **Legacy hooks** — 删除 `useFadeInUp.ts`, `useScaleBounce.ts`, `useBarGrow.ts`，清理 `hooks/index.ts`
- [x] **Schema 对齐** — 确认 `types.ts` 的 `primaryFont + fallbackFont` 是更完善的实现，spec 文档 JSON 示例为简化版
