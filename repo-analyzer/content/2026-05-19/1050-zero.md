# vercel-labs/zero

- **GitHub 地址**：https://github.com/vercel-labs/zero
- **一句话定位**：全球第一个"为 AI Agent 而非人类"设计的编程语言，让代码的写作者从人变成了 Agent
- **编程语言**：C
- **Star 总数**：2319（本期增长：过去 4 天新建，7 天内新星数排名第 2）
- **许可证**：Apache License 2.0
- **官方简介**：The programming language for agents
- **使用领域标签**：编程语言、AI Agent、编译器、开发工具
- **核心功能**：
  1. **Agent 优先的语言设计**：极小且规整的语法面，Agent 可以通过示例、文档和编译器反馈快速学会，不需要人类阅读习惯的妥协
  2. **深度标准库**：将常见能力内置于统一的库 API 中，而非像传统语言那样依赖 npm/pip 生态的碎片化依赖栈
  3. **确定性工具链**：诊断信息、依赖图、体积报告、修复方案等全部以结构化 JSON 输出，Agent 可直接解析和执行
  4. **全平台编译**：支持编译为原生可执行文件（如 linux-musl-x64），单文件部署，无运行时依赖
  5. **Web 路由内置**：`zero routes` 命令可直接输出 Web 服务的路由结构，Agent 无需猜就能理解服务架构
- **目标用户**：AI Agent 开发者、自动化脚本构建者、探索"人机协作编程"范式的研究者和工程师
- **快速上手**：
  ```bash
  curl -fsSL https://zerolang.ai/install.sh | bash
  export PATH="$HOME/.zero/bin:$PATH"
  zero check examples/hello.0   # 检查语法
  zero run examples/add.0       # 运行程序
  zero graph --json examples/systems-package  # 输出结构化依赖图
  ```
- **应用场景举例**：
  - 让 AI Agent 自主学习一门新语言并编写可编译的程序，测试 Agent 的"语言理解"能力
  - 在 CI/CD 中用结构化工具链替代 grep/regex 解析编译错误，让自动化流水线更可靠
  - 构建完全由 Agent 驱动的"无人类介入"的软件开发流水线
