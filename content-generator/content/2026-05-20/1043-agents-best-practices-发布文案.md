## 视频简介/描述

### B站简介
agents-best-practices 是一个面向 AI Agent 系统设计的开源知识库，MIT 协议发布。它提出了一个核心架构原则：模型只负责提出动作，真正的验证、授权、执行和记录由外围 Harness 完成。仓库包含 15 份参考文档，从 MVP 蓝图到上线 Checklist，覆盖工具权限分级、上下文压缩、提示词缓存、安全评估等关键环节。项目以 Agent Skill 形式发布，兼容 Codex 和 Claude Code，一行命令安装。无论你构建的是编程 Agent 还是业务流程 Agent，这套运行时纪律都适用。

GitHub: https://github.com/DenisSergeevitch/agents-best-practices

### 小红书正文
Agent 系统最容易踩的坑不是 prompt 写不好，而是运行时纪律没建好。

这个开源项目把"模型提议、Harness 执行"的架构范式拆成了 15 份实战文档，工具权限分级、上下文压缩策略、上线 Checklist 全都有。支持 Codex 和 Claude Code，一行命令安装。

正在搭 Agent 的朋友建议收藏。

GitHub: DenisSergeevitch/agents-best-practices
