## 视频简介/描述

### B站简介
Vercel Labs 开源了 Zero——一个从零开始为 AI Agent 设计的编程语言。它不是在现有语言上套壳，而是从编译器、标准库到工具链全部重建：语法极小且规整让 Agent 可以快速学会，工具链全结构化 JSON 输出让 Agent 直接解析和执行，标准库深度内建减少依赖搜索。项目用 C 编写，Apache 2.0 开源，4 天拿下 2300 Star。目前还在 pre-1 阶段，语法和 API 会持续变更。"Agent-first language"这个理念本身对自动化流水线和 Agent 工具链设计有参考价值。GitHub: github.com/vercel-labs/zero

### 小红书正文
Vercel 出了个叫 Zero 的编程语言，不是给程序员用的——是给 AI Agent 用的。语法极简让 AI 一学就会，编译器输出全 JSON 方便直接解析，还内置 Web 路由分析。目前还在早期阶段，但"语言的第一用户是 AI"这个思路挺有意思。github.com/vercel-labs/zero
